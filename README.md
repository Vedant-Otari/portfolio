# Portfolio Website

Personal portfolio showcasing projects, skills, and experience as a Software Developer.

## 🚀 Tech Stack
- React.js
- Tailwind CSS
- Framer Motion
- JavaScript

## 📌 Features
- Responsive UI
- Project showcase with details modal
- Smooth animations
- Skills and experience sections
- Contact links

## 🛠️ Setup

```bash
npm install
npm start
```
